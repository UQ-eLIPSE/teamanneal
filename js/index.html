Notes

  • Messages and errors are not presented to UI; please monitor console during execution
  • Code designed for use with browsers released after August 2016 with good ES6 support
  • Local execution under file: URI only appears to work in Firefox; please use http: URI scheme otherwise
  • Memory usage is high (~50-100MB) due to code recording ALL objects/arrays during anneal history for debugging purposes; should normally be around ~5MB otherwise (for source record size ~1000 entries)
  • Output CSV may have reordered columns
  • Group names are generated in fixed format (Pm::Gn) and are under column "__groups"
  • Not all features in original TeamAnneal constraints file supported (name formats, only last level used, etc.)
